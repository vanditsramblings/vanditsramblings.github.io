The clear line shortcut **Ctrl + U** works for password as well.


A quick rundown of similar commands : 

1. Clear word to the left of cursor : **Ctrl+W** 
    > Example : [Example text **|** cursor is here ]
    > Ctrl+W will clear "text"
2. Clear everything after cursor : **Ctrl+K** 
    > Example : [Example text **|** cursor is here ]
    > Ctrl+K will clear " cursor is here"
3. Clear everything before cursor : **Ctrl+U** 
    > Example : [Example text **|** cursor is here ]
    > Ctrl+U will clear "Example text"
4. Paste last cleared text : **Ctrl+Y** 
    > Example : [Example text **|** cursor is here ]
    > Ctrl+Y will paste "Example text"


-- 

This works for Vim as well , as it uses similar keybindings. I am not a VIM user myself, but it is good to know for those **ssh** sessions.
